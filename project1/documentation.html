The Process

To create this site, I began by coming up with content and writing it in markdown. I then marked up this content with HTML. Next came styling with CSS, which demanded the most attention. I initially started styling the landing page, and mistakingly did this with desktop in mind first. I completed the styling for the page, and then worked my way down in screen size to style it appropriately for tablets and phones. After doing this, I edited the CSS for a mobile-first approach, and made an effort to style the rest of the pages in this manner.
